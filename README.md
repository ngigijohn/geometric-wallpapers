# Geometric Wallpaper Studio

A premium, fully interactive generative art engine that creates stunning mathematical wallpapers in your browser.

## Features

- **6 Pattern Engines**: Low Poly (Delaunay), Voronoi Cells, Sacred Geometry, Bauhaus/Memphis, Isometric 3D, Flow Field Waves
- **Advanced Color System**: 10 curated palettes + HSL Color Harmony Generator
- **Interactive Canvas Sculpting**: Click to add vertices, drag to reshape meshes
- **Live Screensaver Morphing**: Organic drift animations with speed/amplitude controls
- **High-Res Export**: PNG up to 4K UHD (3840×2160), Scalable Vector SVG
- **Multi-Aspect Ratios**: Desktop 16:9, Mobile 9:16, Tablet 4:3, Square 1:1
- **Favorites Gallery**: Save & restore designs via localStorage
- **Undo/Redo**: 30-step history stack

## Tech Stack

Pure HTML5 Canvas + Vanilla CSS + Vanilla JavaScript. Zero dependencies, zero build step.

## Usage

Open `index.html` directly in any modern browser or visit the [live demo](/).
